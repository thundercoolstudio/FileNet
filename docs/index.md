# 欢迎使用FileNet
**您可以使用此项目来快速的构建一个属于您个人的网盘网站（如果您想要将此网盘网站暴露在公网中，您或许需要一个内网穿透软件，您可以使用cpolar进行此操作，我们专门对其进行了适配）**

## 特点
* **简易**
* **轻量**
* **跨平台支持（鸿蒙除外）**

##环境要求
* **`Windows7+ / Linux2.x + / Android Termux / ...`**
* **`Python3.12+`**
* **`Flask3.0.3`&`Flask Login`**

## 快速使用
**仅仅需要做如下操作：**
```bash
~$ cd Your_Project_Url/
~$ python main.py
```
